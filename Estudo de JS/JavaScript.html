Variáveis

usa-se as palavras "var" e "let" para declarar as variáveis

IMPORTANTE: Há diferença entre letras minúsculas e letras maiúsculas para a declaração das variáveis.

Tipos de variáveis: Number (pode ser subdividida em Infinity e NaN); string; boolean; null; underfined; object (array); function.

As variáveis podem começar com letra, $ ou _ . É possível usar acentos e símbolos. Não podem começar com números. É possível usar letras ou números. Não podem conter espaços

Para saber o tipo da variável vc pode declarar a variável e usar o comando "typeof"

Concatenação

para concatenar em JS é usado o sinal de mais "+". Se for um number + number será feita uma adição. Se for uma string + string será feita uma concatenação

Para concatenar uma variavel em meio à uma string pode usar o ${ }. Exemplo: 'O aluno ${nome} com ${idade} anos tirou a nota &{nota}'

TAGs do JavaScript

  • document.write() - escreve na tela o que colocar entre os parenteses
  • .toUpperCase() - coloca todas as letras em maiúsculo
  • .toLowerCase() - coloca todas as letras em minúsculo
  • // ou /* */ - Utilizado para realizar comentários
  • Number.parseInt(n) - converte uma string para número inteiro
  • Number.parseFloat(n) - converte uma string para número real
  • Number - O computador decidirá para qual tipo de número irá converter
  • .toFixed(n) - fixa a quantidade de casas decimais desejadas, onde "n" é a quantidade de casas desejadas
  • .replace ("X","x") - troca uma coisa por outra, neste exemplo o "X" será substituído pelo "x"
  • .toLocaleString('pt-BR, {style: 'currency', currency: 'BRL'}') - Esse comando vai converter o número em dinheiro (real)

Operadores aritméticos

Os operadores aritméticos são: + ; - ; * ; / ; % (divisão que mostra o resto da divisão inteira Ex: 5 % 2 = 2 e resta 1, ele mostrará o resto, neste caso o 1); e ** (potência)

A precedência entre os operadores aritméticos da matemática também é respeitada aqui.

Abreviação de operação: n = 3 -> n = n + 4 pode ser abreviado em n += 4

Operador de incremento: x++ é a mesma coisa que x = x + 1 ; x-- é a mesma coisa que x = x - 1

Operadores relacionais

Os operadores relacionais são: > ; < ; >= ; <= ; == (igual); != (diferente)

O resultado de uma operação com operadores relacionais será sempre um valor booleano

Operadores lógicos

Os operadores lógicos são: ! ; &&; || ; sendo exclamação, conjunção e disjunção respectivamente.

Precedência dos operadores: Primeiro é o operador aritmético, depois os operadores relacionais, depois os operadores lógicos

Precedência entre os operadores lógicos: Primeiro o "não", depois o "e", depois o "ou"

Operador ternário

Os operadores ternários são: "?" e ":"

São usados para realizar testes lógicos e retornar um valor

Sintaxe: teste ? true : false

Ex: média >= 7.0 ? "Aprovado" : "Reprovado"

O caso acima retornará "Aprovado" se a média do aluno for maior ou igual a 7.0 e "Reprovado" se a média do aluno for inferior a 7.0

Condição múltipla

switch (expressão) {case 1: break case 2: break case 3: break default:(será feito se nenhuma das case foram feitas) break }

Não pode esquecer de colocar o break no final de cada case

Exemplo:

            var agora = new Date()
            var diaSem = agora.getDay()

            switch(diaSem) {
                case 0:
                console.log("Domingo")
                break

                case 1:
                console.log("Segunda-feira")
                break

                case 2:
                console.log("Terça-feira")
                break

                case 3:
                console.log("Quarta-feira")
                break

                case 4:
                console.log("Quinta-feira")
                break

                case 5:
                console.log("Sexta-feira")
                break

                case 6:
                console.log("Sábado")
                break

                defaul:
                console.log("[ERRO] Dia inválido")
                break
            }

Lembrando que para o JS a semana começa no dia 0 - Domingo e termina no dia 6 - Sábado.

Variáveis compostas

Variáveis simples: só conseguem armazenar um valor por vez

Variáveis compostas: devem ser capazes de armazenar vários valores em uma mesma estrutura

As variáveis compostas são formadas por array, cada array é formada por elementos e esses elementos possuem um valor e um índice

Ex: let num = [5, 8, 20]

É possível acrescentar elementos na array

Exemplo: num [3] = 6 ou pode usar o num.push [7] neste último caso o JS colocará o número 7 na última vaga da array não importa o tamanho dela

para saber o tamanho da array usa-se o num.length

o comando num.sort() colocará todos os elementos em ordem crescente

O comando num.indexOf(x) retornará o índice do número que vc colocou no lugar do x, ou seja, se for colocado num.indexOf(3) será retornado em qual posição da array está o número 3
